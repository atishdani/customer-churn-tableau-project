# Customer Churn Analysis Dashboard

## Project Overview
This project analyzes customer churn behavior using Tableau to identify patterns, trends, and business insights related to customer retention.

The dashboard provides interactive visualizations that help understand:
- Customer churn trends
- High-risk customer segments
- Revenue impact from churn
- Customer demographics
- Subscription behavior

---

## Tools Used
- Tableau
- CSV Dataset
- Data Visualization
- Dashboard Design

---

## Dataset Information
The dataset contains customer-related information including:
- Customer demographics
- Subscription details
- Payment information
- Churn status
- Revenue metrics
